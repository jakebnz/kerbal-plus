Download & Install

Installation is just a few simple steps

Step 1: Install CKAN

CKAN is a mod manager for Kerbal Space Program. You'll need to download and install CKAN to install the Kerbal Plus modpack. Download the latest release from their GitHub releases page.

Download CKAN

Step 2: Download the modpack file

Click below to download the official Kerbal Plus modpack file:

Download Modpack

Step 3: Import into CKAN

Open CKAN and select File -> Install from .ckan. Navigate to where you saved the downloaded file and select it. CKAN will automatically queue all required mods and their dependencies.

Step 4: Apply and Install

Click the "Apply changes" button in CKAN. Wait for the download and installation process to complete. Depending on your internet speed, this may take some time.

Step 5: Launch!

Once CKAN confirms the installation is complete, you can launch Kerbal Space Program and enjoy.
